# Patches

Patch manifests for cross-skill or cross-artifact updates originating in this repo. See `skills/knowledge-base-reference/` companion docs for the patch manifest convention and template.

## Naming

`YYYY-MM-DD-slug.md` — descriptive slug. If multiple patches land same day, append `-NN` suffix.

## When required

- Changes touching files in 2+ skill folders (when iterating on the upstream skills repo)
- Changes to companion artifacts
- Tech debt resolution
- New skill ship
- Cross-skill iteration passes

## When optional

- Single-file edits
- Inbox sweeps
- Routine operational use (capture, synthesis, narrative)
